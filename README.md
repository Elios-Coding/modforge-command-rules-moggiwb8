# DropMultiplier

Server-side Fabric mod adding /multiplier and /mobmulti commands to control block and mob drop multipliers by multiplying item stack counts on pickup.

Generated by **ModForge AI**.

- Minecraft: `1.21.11`
- Loader: Fabric `0.18.4`
- Fabric API: `0.141.3+1.21.11`

## Build

```bash
./gradlew build
```

The compiled `.jar` will be placed in `build/libs/`. GitHub Actions builds it automatically on push and publishes it to Releases.
